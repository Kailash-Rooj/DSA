# Basic Recursion

> A curated collection of **fundamental recursion problems** to build a strong foundation in recursive thinking and problem-solving.

---

## 🗂️ Problems List

### 🔹 Problem 0 — Print Name N Times

**Print a given name N times using recursion.**

```
Input: N = 3 , name = "Ashish"
Output: Ashish Ashish Ashish 
Explanation: Name is printed 3 times.

Input: N = 1 , name = "Ashish"
Output: Ashish 
Explanation: Name is printed once.

```

**File:** `p00.java`

---

### 🔹 Problem 1 — Print 1 to N

**Print numbers from 1 to N using recursion.**
```
Input: N = 4
Output: 1, 2, 3, 4
Explanation: All the numbers from 1 to 4 are printed.
```
**File:** `p01.java`

### 🔹 Problem 2 — Sum of First N Natural Numbers

**Find the sum of the first N natural numbers using recursion.**

```
Input: N=5
Output: 15
Explanation: 1+2+3+4+5=15

Input: N=6
Output: 21
Explanation: 1+2+3+4+5+6=21
```

**File:** `p02.java`

---

### 🔹 Problem 3 — Factorial of a Number

**Find the factorial of a given number using recursion.**

```
Input: X = 5
Output: 120
Explanation: 5! = 5*4*3*2*1
```

**File:** `p03.java`

Input: N=6 arr[] = {10,20,30,40}

### 🔹 Problem 5 — Fibonacci Series up to Nth Term

**Print the Fibonacci series up to the Nth term using recursion.**
```
Example 1:
Input: N = 5
Output: 0 1 1 2 3 5
Explanation: 0 1 1 2 3 5 is the fibonacci series up to 5th term.(0 based indexing)
```

**File:** `p05.java`

### 🔹 Problem 6 — Palindrome String Check

**Check if a given string is a palindrome using recursion.**
```
Example 1:
Input: Str =  “ABCDCBA”
Output: Palindrome
Explanation: String when reversed is the same as string.

Example 2:
Input: Str = “KAILASH”
Output: Not Palindrome
Explanation: String when reversed is not the same as string.
```

**File:** `p06.java`
